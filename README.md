# @tiegz dotfiles

### Installation

#### 1) clone the repo:

``` sh
cd ~
git clone git@github.com:tiegz/dotfiles.git
```

#### 2) add dotfiles to rc script:

#### zsh

``` sh
echo "source ~/dotfiles/init" >> ~/.zshrc
source ~/.zshrc
```

#### bash

``` sh
echo "source ~/dotfiles/init" >> ~/.bash_profile
source ~/.bash_profile
```
